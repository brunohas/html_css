Imagens dinâmica

Para que a imagem abra sem se importar com o tamanho da tela do usuário utiliza-se as tags <picture> para conseguir mudar de imagem (colocar uma menor ou maior), <img> para colocar a "imagem principal" e a tag <source:media:type> para colocar as imagens restantes (abrir uma tag para cada imagem), a ordem é como se fosse uma pirâmide invertida, colocar a tag <img> em último e acima dela colocar as tag <source:media:type> de acordo com a quantidade de fotos e de tamanho (maior em último (tag img) / média no meio (tag source) / pequena acima do outro source (tag source)).

Na hora que abrir a tag <source:media.type> mudar a opção (min-width: ) para (max-width: ) e sempre colocar 50px amais do que a foto que você quer colocar, exemplo: a foto G tem tamanho de 1000px → no source da foto M coloco o max-width: 1050px para que se a tela não suportar a imagem principal = foto G abrir a foto M.

Agora aumente e diminua a janela do navegador para ver as imagens mudando para se adaptar ao tamanho da tela
