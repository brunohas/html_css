Tags HTML

Todas as tags HTML devem estar em < >, e fechar com </ > (existem algums que não precisa).

  • Tags não fechadas
    • <br> → quebra de linha
    • <hr> → linha horizontal
  • Tags de texto
    • <p> → abrir um parágrafo
    • <h1>, <h2>, ..., <h6>. → tags de títulos
    • <wbr> → quebra de palavra
    • <strong> → negrito
    • <em> → itálico
    • <abbr> → abreviação
    • <del> → texto riscado
    • <ins> → texto sublinhado
    • <sup> → texto sobrescrito (x20+3)
    • <sub> → texto subscrito (H2O)
    • <adress> → informar um endereço
    • <bdo> → inverte o texto
    • <bdi> → isola o conteúdo do conteúdo ao redor
    • <mark> → marca o texto
    • <blockquote> → destaca citações
    • <cite> → destaca citação indicando a fonte
    • <q> → destaca citação curta
    • <article> → "artigo" define um conteúdo independente e autocontido (usado em citações)
    • <aside> → indicar informações importantes sobre o conteúdo principal
    • <header> → cabeçalho (colocamos p/ os mecânismos de busca valorizar mais o conteúdo)
    • <main> → conteúdo principal (colocamos p/ os mecânismos de busca valorizar mais o conteúdo)
    • <footer> → rodapé (colocamos p/ os mecânismos de busca valorizar mais o conteúdo)
    • <nav> → "menu de navegação" representa uma seção de navegação em uma página web
    • <div> → define divisões lógicas em uma página web
    • <section> → "sessão" utilizada para agrupar conteúdo relacionado em uma seção lógica de uma página
    • <pre> → definir que é um texto pré-formatado
    • <code> → indica que o texto é um código (deixa todas letras/caracteres do mesmo tamanho)
    • <dfn> → marca um termo que será definido no documento
    • <kdb> → elemento de entrada do teclado, entrada de voz ou qualquer dispositivo de entrada de texto
    • <samp> → identificar o texto que deve ser interpretado como saída de amostra de um programa
    • <var> → identificar uma variável em uma equação matemática ou programa de computador
  • Tags p/ link
    • <a> → cria hiperligação em páginas web, arquivos, endereço de e-mail, entre outros
    • <base> → especificar a URL base ou o local de destino p/ todas as URLs relativas dentro do documento
  • Tags de imagem e objeto
    • <img> → inserir imagens em uma página
    • <map> → define uma área clicável no mapa de imagem (é usado com o elemento <area>)
    • <area> → define uma área clicável dentro da imagem, e opcionalmente, associa-lo com um link (usado dentro de um <map>)
    • <object> → define um contêiner para um recurso externo, como uma imagem, página web, reprodutor de mídia ou app plug-in
  • Tags de listas
    • <ul> → abre uma lista sem ordem
    • <ol> → abre uma lista ordenada
    • <li> → utilizada dentro das tags <ul> ou <ol> para representar um item da lista
    • <dl> → representa uma lista de associação, lista de descrição
    • <dt> → define um termo ou nome em uma lista de descrições
    • <dd> → define a descrição de um termo em uma lista de descrição

    Agora alguns códigos em CSS para estilizar a lista:

    • list-style-type: → conseguimos mudar a bolinha da lista por um emoji, utilizar '\' para adicionar emoji, '\00A0' da um espaço
      (ex: list-style-type: '\2714';)
    • list-style-position: → posição das bolinhas da lista (inside → dentro / outside → fora)
    • columns: → divide a lista em colunas (colocar a qnt de colunas que deseja)

Tags CSS

Todas as tags de estilo devem estar em dentro da tag <style> (ou dentro de um arquivo criado em CSS) ou em uma tag específica que deseja estilizar.

  • color: → cor da letra
  • background-color: → cor de fundo
  • background-image: linear-gradient(); → cria um degradê em linha (to top / to right / to bottom / to left)
  • background-image: radial-gradient(); → cria um degradê em circulo (to top / to right / to bottom / to left)
  • background-image: url(); → coloca uma imagem de fundo
  • background-size: → tamanho da imagem de fundo (auto = padrão / contain = imagem 100% exibida / cover = cobrir)
  • background-repeat: → no-repeat = não repete a imagem / repeat = repete a imagem / repeat-y = repete a imagem em uma linha vertical / repeat-x = repete a imagem em uma linha horizontal
  • background-attachment: fixed; → fixa o fundo em relação à janela de visualização
  • background-attachment: scroll; → fundo rola junto com o elemento
  • font-family: → tipo de fonte
  • font-size: → tamanho da fonte (colocar em px ou em)
  • font-weight: → "peso" da fonte (lighter / normal / bold / bolder)
  • font-style: → normal ou italic (itálico)
  • text-aling: → alinhamento do texto (left / center / right / justify)
  • text-decoration: → decoração do texto (sublinhado = underline / sobrelinhado = overline / riscado = line-through)
  • text-shadow: → sombra na letra (0px 0px 0px cor)
  • text-indent: → espaço do começo do parágrafo (colocar em uns 20px 30px)
  • width: → "cria" uma janela (colocar largura em px)
  • border-radius: → arredonda as pontas do width (colocar em px ou %)
  • box-shadow: → sombra do width (0px 0px 0px cor) / (inset 0px 0px 0px cor) sombra para dentro
  • padding: → "acolchoamento" (borda pro width)
  • margin: auto; → coloca width no meio da tela
  • display: → mudar os modelos de caixas (block = box-level (quebra de linha) / inline = inline-level (não quebra de linha))
  • border-image-source: url() → borda decorada (colocar url da imagem que deseja)
  • border-image-slice: → "fatiar" a imagem (colocar 20)
  • border-image-repeat: → repeat → repete a imagem / stretch → puxa/distorce a imagem
  • line-height: → espaço entre as linhas de um parágrafo
  • min-height: → altura mínima
  • max-heigth: → altura máxima
  • transition-duration: → serve para dar uma transição (normalmente entre 0.5s e 1s)
  • position: → posicionar pequenos detalhes do layout:
    • static → seguir o fluxo comum da página
    • relative → passa a aceitar as propriedades top, rigth, bottom, left (ex: {position: relative; top: 20px; right: 20px})
    • absolute → posicionar qualquer elemento de acordo com o elemento pai que tenha um position diferente de static
    • fixed → parecido com o absolute, ele passa a referenciar a janela do navegador
  • list-style-type: → conseguimos mudar a bolinha da lista por um emoji, utilizar '\' para adicionar emoji, '\00A0' da um espaço
    (ex: list-style-type: '\2714';)
  • list-style-position: → posição das bolinhas da lista (inside → dentro / outside → fora)
  • columns: → divide em colunas (colocar a qnt)
  • transform: → aplica uma alteração de transformação (scale() = aumentar/diminuir / perspective() = perspectiva / rotate() = rodar / translate() = mexe no componente)

    • Existem algumas ShortsHands que são um conjunto de algumas tags:

  • font: → (font-style: → font-weight: → font-size: → font-family)
  • border: → (border-width: → border-style: → border-color:)
  • padding: → (padding-top: → padding-right: → padding-bottom: → padding-left:)
  • margin: → (margin-top: → margin-right: → margin-bottom: → margin-left:)
  • outline: → (outline-width: → outline-style: → outline-color:)
  • border-image: → (border-image-source: url() → border-image-slice: → border-image-repeat:) criar uma tag border: para ter uma borda
  • background: → (color: → image: → position: → repeat: → [size]: → attachment: )
